# Desktop Target

Проект сейчас ведется как desktop-only demo.

Текущая минимальная ширина интерфейса: 1280px. Это намеренное временное ограничение для этапа верстки и scroll-анимаций. Мобильная адаптация не считается частью текущей рабочей области, пока пользователь явно не вернет ее в задачу.

Базовая минимальная планка для проверки производительности: офисный ноутбук 2018-2020 годов, 8 GB RAM, интегрированная графика.
