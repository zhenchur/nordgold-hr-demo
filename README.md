# Nordgold HR Demo

Presentation demo for the Nordgold HR careers landing page.

## Stack

- Vite
- TypeScript
- GSAP / ScrollTrigger
- Lenis

## Run Locally

```bash
npm install
npm run dev
```

The dev server starts on `127.0.0.1`; Vite will print the exact port.

## Build

```bash
npm run build
```

Generated folders such as `node_modules`, `dist`, `output`, and local logs are ignored by git.

## GitHub Pages

Production builds use the GitHub Pages base path `/nordgold-hr-demo/`.
After `npm run build`, preview the production build at:

```bash
npm run preview
```

Then open `http://127.0.0.1:4173/nordgold-hr-demo/`.
