# Nordgold HR Demo Fixlist

Дата старта: 2026-06-09

Правило ведения:
- Номера стабильные и видны прямо в строке чекбокса.
- Пользователь ставит галочку `[x]` у пунктов, которые нужно взять в работу.
- Codex не использует чекбокс как маркер выполнения. Чекбокс означает только "взято в работу".
- Выполнение Codex отмечает отдельной строкой `Статус: выполнено`.
- Если чекбоксы неудобно редактировать, пользователь пишет номера в чат, а Codex сам отмечает эти пункты `[x]` перед началом работы.
- Коммит/пуш делаем только по отдельной команде пользователя.
